Student Projects

Our students bring innovation to life through practical projects that connect classroom learning with real-world applications. Each project showcases teamwork, creativity, and technical excellence across different computing disciplines.

Fire Alarm System using ESP32 & Telegram Alerts

A smart fire detection and alert system using ESP32 microcontroller, capable of sending instant notifications via Telegram for real-time safety response.

Team: Bhat Bikram Singh, Chand Anup, Machamasi Nirjala, Jaiswal Anand Bihari, Nath Rashmi

Batch: Spring 2023

Electricity Bill Management System

Course: Python Programming

A Python-based application that automates electricity bill generation, consumption tracking, and monthly reports for residential users.

Team: Bhat Bikram Singh

MazeEscape: A Puzzle Adventure Game

Batch: Spring 2023

Solar System 3D Simulation
