# Smart Computing Department Website

A responsive frontend website developed as the final project for the Web Application Development course. This project presents a modern departmental website for the Department of Smart Computing, providing information about academic programs, faculty members, laboratories, student projects, events, and contact details.

# Features

* Responsive homepage
* About the department
* Curriculum information
* Events and activities
* Student projects showcase
* Laboratory information
* Faculty page
* Contact page
* Clean and user-friendly interface

# Technologies Used

* HTML5
* CSS3


# Project Structure

home.html                 # Homepage
about.html                # About the department
curriculum.html           # Curriculum details
events-activities.html    # Events and activities
s_projects.html           # Student projects
labs.html                 # Laboratories
faculty.html              # Faculty information
contact.html              # Contact page
style.css                 # Website styling
images/                   # Images used throughout the website

# How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open home.html in any modern web browser.

No additional software or installation is required.

# Project Objective

The objective of this project is to design a responsive departmental website that demonstrates frontend web development skills using HTML and CSS. The project was completed as part of the Web Application Development course.


# Note

This repository contains only the frontend implementation. It does not include backend functionality or a database.

Some images used in the original academic submission have been removed or replaced in this public repository where appropriate.
